## Getting Started:
### Things needed:
* A raspberry pi 3B
* A Neural Compute Stick - (NCS)
* A pan/tilt bracket - (3D print)
* Two Servos - (SG90)
* A GPIO expansion board
* Pi Camera or USB Webcam
